# PlugQuiche

HTTP/3 adapter for [Phoenix](https://www.phoenixframework.org/) and [Plug](https://github.com/elixir-plug/plug) via [Cloudflare quiche](https://github.com/cloudflare/quiche).

Powered by [quiche_erl](https://github.com/BobbieBarker/quiche_erl) for QUIC transport.
